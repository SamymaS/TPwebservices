## 🎯 Démarrage rapide

### 1. Backend

```bash
cd backend
npm install

# Créer .env avec vos clés Supabase
npm start
```

### 2. Frontend

```bash
cd frontend
npm install
npm run dev
```

### 3. Tests

Importez les collections Postman dans `../postman/`
